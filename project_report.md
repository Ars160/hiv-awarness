# Project Report: HIV Awareness Website

## 1. Project Overview

This project is a single-page web application designed to raise awareness about HIV. It provides users with information about HIV, including statistics, news, testing information, and support resources. The website is designed to be informative and easy to navigate, with a clear and modern user interface.

## 2. Key Features

The website consists of several sections, each dedicated to a specific topic:

*   **Header:** A navigation bar for easy access to different sections of the page.
*   **Hero Section:** A prominent introductory section that likely contains a key message and a call to action.
*   **Courses:** A section to provide educational courses or materials.
*   **Statistics:** A section displaying key statistics related to HIV.
*   **Video Section:** An embedded video for informational purposes.
*   **Testing Information:** A dedicated section with information on where and how to get tested.
*   **News:** The latest news and updates related to HIV.
*   **Support:** Information about support groups and resources.
*   **Team Section:** A section to introduce the team or key people involved.
*   **FAQ/Features:** A section to answer frequently asked questions.
*   **Contact:** A form or contact details for users to get in touch.
*   **Footer:** Contains additional links and information.

## 3. Technologies Used

The project is built using modern web technologies:

*   **Frontend Framework:** React.js
*   **Styling:** Tailwind CSS
*   **Routing:** React Router
*   **Icons:** Heroicons, React Icons
*   **Maps:** Yandex Maps
*   **Analytics:** Vercel Speed Insights
*   **Build Tool:** Create React App

## 4. Project Structure

The project follows a standard React application structure:

*   `public/`: Contains the main HTML file and static assets like images and icons.
*   `src/`: Contains the main application logic.
    *   `assets/`: Contains images used in the application.
    *   `components/`: Contains reusable React components for different sections of the website.
    *   `App.js`: The main component that assembles the different sections of the page.
    *   `index.js`: The entry point of the application.
    *   `index.css`: Contains global styles.
*   `package.json`: Defines the project's dependencies and scripts.
*   `tailwind.config.js`: Configuration file for Tailwind CSS.
